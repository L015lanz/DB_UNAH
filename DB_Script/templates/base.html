En este momento estamos en la aplicacion web DB_UNAH.

Es una aplicacion la cual pretende mostrar cada parte de la base de datos diseñada y construida para simular el funcionamiento de los sistemas de Admisiones, Matricula y Campus Virtual de la Universidad Nacional Autónoma de Honduras. Se incluyen Procedimientos Almacenados, Secuencias, Tablas y Datos incrustados en la misma.

Esta es la version 1.0.0, en futuras versiones se incluiran otros proyectos y mejoras a esta aplicacion web
